\chapter{C4-ДЕКОМПОЗИЦИЯ АРХИТЕКТУРЫ}

В~приложении представлена текстовая C4-декомпозиция архитектуры
ИИ-методиста. На~этапе черновика она заменяет финальные визуальные
диаграммы и~фиксирует состав системы в~форме, достаточной
для~архитектурного обсуждения.

\section*{Г.1 --- Системный контекст (C4 Level~1)}

\begin{itemize}
  \item \textbf{Разработчик курса / HR-специалист}: загружает документ,
    запускает подготовку черновика, ревьюит структуру курса;
  \item \textbf{BFF School}: маршрутизирует запросы веб-клиента;
  \item \textbf{ИИ-методист}: выполняет обработку документа, анализ,
    подготовку структуры и~черновых материалов курса;
  \item \textbf{LMS Adapstory}: принимает итоговый курс и~публикует его;
  \item \textbf{MinIO}: хранит исходные документы и~медиа;
  \item \textbf{Ollama / GPU-узел}: обслуживает локальные LLM и~модели мультимедиа;
  \item \textbf{Qdrant}: хранит векторный индекс для~поиска релевантных фрагментов.
\end{itemize}

\section*{Г.3 --- Компоненты (C4 Level~3)}

\begin{itemize}
  \item \textbf{Document parser}: преобразует PDF/DOCX/PPTX/XLSX в~текст;
  \item \textbf{Chunker + embedder}: формирует индексируемые фрагменты;
  \item \textbf{CourseStructureAgent}: генерирует структуру курса;
  \item \textbf{ContentCreatorAgent}: формирует тексты уроков;
  \item \textbf{FactCheckAgent}: проверяет фактические утверждения;
  \item \textbf{Review / HITL layer}: обеспечивает контроль с~участием человека;
  \item \textbf{Exporter}: публикует результат в~LMS Adapstory.
\end{itemize}
