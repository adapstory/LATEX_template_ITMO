\section*{Дополнительные открытые источники}

В~качестве расширенного референсного массива собраны дополнительные
открытые учебные материалы:

\begin{itemize}
  \item \textbf{Anthropic Academy / Skilljar}: 16~курсов;
  \item \textbf{OpenEdu.ru}: 5~курсов российских вузов;
  \item \textbf{HuggingFace Learn}: 12~курсов по~ML/AI;
  \item \textbf{Telegram-курсы}: 2~бота-симулятора как примеры
    ролевого сценарного обучения.
\end{itemize}

\section*{Оценочные артефакты следующей итерации}

На~текущем этапе полностью подготовлены:
\begin{itemize}
  \item ядровой корпус из~16 документов;
  \item набор из~30 QA-пар для~RAG-оценки;
  \item набор из~100 утверждений для~оценки факт-чекинга.
\end{itemize}

Экспертный эталонный корпус для~слепого сравнения конфигураций B1--B4
оформлен как~следующий этап исследования и~в~текущем черновике
не~представляется как завершённый измеренный результат.
