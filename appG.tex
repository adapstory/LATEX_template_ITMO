\section*{Дополнительные источники}

В~качестве дополнительных материалов для~расширенных экспериментов
собраны курсы из~открытых платформ:

\begin{itemize}
  \item \textbf{Anthropic Academy} (Skilljar): 16~курсов, 424~урока,
    48~YouTube-ссылок~--- профессиональные курсы по~работе с~Claude
    и~API Anthropic;
  \item \textbf{OpenEdu.ru}: 5~курсов российских вузов (ВШЭ, ИТМО, МГУ)~---
    русскоязычные программы с~соответствием ФГОС;
  \item \textbf{HuggingFace Learn}: 12~курсов по~ML/AI~---
    англоязычные технические программы;
  \item \textbf{Telegram-курсы}: 2~бота-симулятора~--- примеры ролевого
    обучения с~выбором решений (842 и~1113~сообщений).
\end{itemize}

\section*{Ground truth}

Для каждого из~15~документов основного корпуса два независимых
эксперта-методиста (стаж разработки eLearning-курсов~$\geq$~5~лет)
составили эталонную структуру курса с~разметкой учебных целей
по~шести уровням таксономии Блума.
Согласованность экспертов оценена коэффициентом Коэна~$\kappa$
(раздел~2.1 основного текста).
Эталоны хранятся в~JSON-формате (схема \texttt{CourseBlueprint})
и~CSV (разметка Bloom's уровней).
